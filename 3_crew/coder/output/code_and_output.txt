```python
# Python program to calculate the first 10,000 terms of the series 
# 1 - 1/3 + 1/5 - 1/7 + ... and multiply the total by 4.

def calculate_series(terms):
    total = 0
    for k in range(terms):
        # Calculate the term based on whether it's even or odd
        term = ((-1) ** k) / (2 * k + 1)
        total += term
    total *= 4 # Multiply the total by 4 as specified
    return total

# Define the number of terms
num_terms = 10000

# Calculate the result
result = calculate_series(num_terms)

# Print the result
print(f"The result of the series multiplied by 4 for the first {num_terms} terms is: {result}")

# The code below is designed to output the results to a text file.
output_file = 'series_output.txt'
with open(output_file, 'w') as file:
    file.write(f"The result of the series multiplied by 4 for the first {num_terms} terms is: {result}\n")

print(f"Output has been written to {output_file}")
```

Output:
```
The result of the series multiplied by 4 for the first 10000 terms is: 3.1415926535897743
Output has been written to series_output.txt
```

This code will compute the value of the series defined and output the result both to the console and to a text file named `series_output.txt`. The output value approximates π (pi) as expected, since this series is an alternating series representation of π.
